A Sprite component renders an sprite on the screen

Parameters
pathPath to the image (png or jpg)
pixelScaleAmount of pixels in image used per 1 pixel on screen (value of 2 will half the image size on screen)
imageSourceSelection of image to render (default uses entire image (Rect2(Vector2(0,0), Vector2(0,0)))
offsetPositionA starting value to offset the sprite position
startingRotationA starting rotation applied to the sprite
pivotPointCustom pivot point in raw pixel value of point in image (default uses center of image (Vector2(0,0))
colorColor used to blend with image (default white with no effect)
